Case Study: FindAddis

Problem

Local businesses in Addis Ababa struggled with online visibility and customer reach.

Solution

FindAddis was developed with real-time search, interactive maps, and business profiles to enhance local visibility and user convenience.

Architecture

  • JavaScript, Node.js, Express, MongoDB, ArcGIS
  • Interactive map integration
  • Real-time search

Tech Stack

  • JavaScript, Node.js, Express, MongoDB, ArcGIS

Database Schema

{
  business_id: ObjectId,
  name: String,
  address: String,
  category: String,
  location: { lat: Number, lng: Number },
  profile: String
}

Challenges

  • Integrating ArcGIS maps
  • Optimizing real-time search
  • Ensuring data accuracy

Performance Considerations

  • Fast search response
  • Efficient map rendering

Refactor Ideas

  • Improve map UI
  • Expand business categories

Impact

Helped 100+ businesses gain new customers and improved user convenience citywide.
